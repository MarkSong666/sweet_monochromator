蔗糖单色仪项目 README
# 项目概述
本工具基于旋光理论构建光谱传输模型，搭配交互式可视化界面，支持：
-实时调节光学仿真参数，动态刷新光谱
-自动计算峰值波长、半高宽 (FWHM)、边模抑制比 (SMSR)
-多条仿真曲线叠加对比分析
-差分进化算法自动寻优，精准匹配目标特征波长

# 界面功能详解
1. 绘图主区域
-黑色实线：当前参数下实时仿真光谱
-红色圆点：识别到的光谱峰值位置
-红色虚线：峰值对应的半高宽 (FWHM) 标注线
-顶部标题栏：实时显示峰值波长、FWHM 数值、SMSR 边模抑制比
2. 参数调节滑块（支持手动输入 + 拖动）
参数名称	                    物理含义	                      取值范围	    默认值
Concentration	               蔗糖溶液浓度	                      0 ~ 2	        0.7
Theta1 / Theta2 / Theta3	   偏振光学元件旋转角度	              0° ~ 360°	     0
n1 / n2 / n3	               相当于0.7g/ml, 0.2m旋光管节数	  0 ~ 10	     1
拖动滑块或在右侧输入框填数值，光谱会即时刷新
3. 快捷功能按钮
-Add Curve：保存当前光谱曲线，自动分配配色，用于多组参数对比
-Clear All：一键清空所有叠加的对比曲线，恢复初始界面
4. 智能自动优化模块
-Target nm 输入框：填写需要匹配的目标特征波长（单位：nm，默认 550）
-Optimize 按钮：启动差分进化算法全局寻优
-底部进度条：实时展示优化计算进度
-优化完成后：自动同步更新所有角度、系数滑块，输出最优匹配光谱

# 标准使用流程
1. 手动调试仿真
-运行脚本打开交互窗口
-调整浓度、偏振角、模型系数
-观察峰值、FWHM、SMSR 实时变化
-满意参数点击Add Curve留存曲线，方便对照
2. 算法自动寻优
-在Target nm输入目标波长（如 532、600）
-点击Optimize，等待进度条走完
-程序自动更新参数，生成贴合目标波长的最优光谱

# 常见问题
1. 报错找不到 spec_source.csv
检查文件夹层级，必须保证脚本同级../data/下存在该 csv，列名包含wavelength_nm与intensity。
2. 点击优化后界面卡顿无响应
算法迭代计算属于正常现象，请勿重复点击按钮，等待进度条加载完毕即可。
3. 无峰值、不显示 FWHM/SMSR
当前参数下光谱对比度不足，适当调整浓度、偏振角度，强化峰值特征。
4. 文本框输入数值无效
输入值超出预设区间（角度 0~360、系数 0~10），系统会自动钳位到合法范围。

# 注意事项
1. 优化运算期间请勿关闭窗口，否则直接中断寻优。
2. 对比曲线建议控制在 5 条以内，避免界面卡顿。
3. 自定义替换光源光谱时，csv 文件格式、列名不可修改。
